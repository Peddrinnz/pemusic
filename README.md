# peMusic 🎵

peMusic é um projeto desenvolvido como atividade acadêmica com o objetivo de praticar desenvolvimento web utilizando HTML, CSS e JavaScript.  

A aplicação simula um player de música inspirado em plataformas de streaming como o Spotify.

## Objetivo do projeto

O objetivo deste projeto é aplicar conceitos fundamentais de desenvolvimento web, incluindo:

- Estruturação de páginas com HTML
- Estilização com CSS
- Interatividade com JavaScript
- Manipulação do DOM
- Organização de componentes de interface

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)
